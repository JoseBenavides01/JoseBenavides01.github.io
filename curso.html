Computer Science

Cinecias de la Computacio es el futuro del mundo, es la transformacion de lo físico a lo digital, hacer tareas comunes en procesos computacionales para automatizar acciones y ayudar a la velocidad de creacion de nuevas cosas.

Es crear el nuevo mundo, hacer el mundo virtual comodo y facil de manejar para todos.

Arquitectura de computadores

Es la estructura base que hace a un computadora, una computadora, la cpu, la ram, etc, todos estos componentes son parte de la arquitectura de un computador.

Sistemas Operativos

Es lo que facilita la comunicacion entre humano y computador, windows, mac, linus y otros son lo que hacen que se vea mas comodo para nosotros lo que para cunado hacemos alguna ccion en la computadora.

Redes

Es la comunicacion de nodos a travez de ondas electromacneticas o clabes para pasar informacion de un lado a otro.

La Web

La Web es una de las formas mas comunes de mandar, recibir y editar contenido a travez de la red, y hay tres tipos

- La web 1.0

- La web 2.0

- La web 3.0

La primera de estas es para recibir informacion, la segunda es para recibir y mandar informacion, y la tercera es para recibir, mandar y editar informacion, que nos nos sirve para poder hacer clases en linea, la web3 nos orfese el mundo del streaming, de las crypto, de las blockchains y otras cosas más.
